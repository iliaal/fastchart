fastchart

Native C PHP extension. 26 chart families + 2D barcode and QR. SVG-canonical pipeline; raster output via plutovg + libpng / libjpeg-turbo / libwebp.

Side-by-side SVG / PNG / JPEG / WebP renders for every chart family. Source PHP shown above each row.

Build instructions, install via PIE (pie install iliaal/fastchart), CHANGELOG, license.

Example cookbook →

One self-contained PHP script per chart variant. Copy, paste, edit. The same scripts drive the gallery.
